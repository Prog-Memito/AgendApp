# AgendApp.
Presentemente, el tiempo que demanda el realizar una reserva médica es altamente inconveniente, sobre todo para consultas. Mientras que sus únicas alternativas no presenciales son llamadas telefónicas - que pueden extenderse a excesivos minutos en espera - o formularios online, que requieren dar toda la información de identificación del paciente desde cero.

Para agilizar este proceso, se presenta la idea de una aplicación asociada al servicio de CESFAM, que reconozca el perfil de paciente y a través de este portal sea posible agendar citas médicas a plazo inmediato.
Como función secundaria, la aplicación permitirá la acción de un segundo tipo de usuario, funcionario de CESFAM, mediante la que se recogerán datos administrativos.

# ¿Cómo se usa?
AgendApp es una aplicación intuitiva y fácil de usar. Para comenzar, el usuario Paciente debe iniciar sesión; en caso de no poseer una cuenta, deberá registrarse utilizando su RUT para verificar que se encuentre en el sistema. Si el RUT no está registrado en el SISTEMA, no podrá acceder a las funciones de la aplicación. Una vez validado, el usuario podrá completar sus datos personales para habilitar todas las herramientas.
Dentro de la aplicación, el usuario Paciente encontrará tres secciones principales: "Agendar Hora", "Mis Horas", "Mi Perfil".

Agendar Hora: Al pulsar este botón, el usuario deberá elegir la especialidad (tipo de servicio) requerida: Control Médico; Odontología; Psicología. Tras seleccionar la especialidad, aparecerá un listado de las horas disponibles con sus respectivos profesionales en una ventana de 1 a 2 días. En este paso, el usuario podrá elegir el tiempo de antelación para su recordatorio. Una vez confirmada la fecha y hora de la cita, se enviará un correo electrónico con los detalles del agendamiento: fecha, hora, profesional y box correspondiente, ademas de guardar la informacion para el usuario en la misma App. Por defecto, el sistema enviará una notificación al correo avisando una hora antes de la cita médica.

Mis Horas: Aquí el usuario podrá visualizar sus citas vigentes y el historial de agendamientos pasados. Al seleccionar una hora específica, se mostrarán los detalles y se permitirá modificar el tiempo del recordatorio según su preferencia.
Aqui existe la opcion de cancelacion de cita, que dara un aviso de advertencia antes de confirmar la accion.

Mi Perfil: donde se podrán editar datos como el correo electrónico y la contraseña. Cabe destacar que el usuario no podrá modificar su nombre ni su apellido, ya que estos datos vienen predefinidos por el sistema.
