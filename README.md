# AgendApp.
Es una aplicacion movil creada para ayudar al usuario o pacientes del area de la Salud CESFAM permitiendole agendar horas solamente para controles medicos y psicologo (kinesiologo a futuras versiones) dentro de un rango de 1 a 2 dias, al usuario le mostrara en pantalla los dias y horas disponibles y el nombre del profesional encargado, al asignar su dia, hora y profesional al usuario se le enviara al correo electronico con los datos de la fecha, hora, profesional y box asignado asi mismo marcando como evento en el calendario del dispositivo y activando un recordatorio cuando falte 1 hora para su hora medica.
# Instalación
El método de instalación es muy sencillo, ya que se podrá descargar a través de tiendas virtuales como Google Play Store (para dispositivos Android) o mediante el repositorio oficial del proyecto
# ¿Cómo se usa?
AgendApp es una aplicación intuitiva y fácil de usar. Para comenzar, el usuario o paciente debe iniciar sesión; en caso de no poseer una cuenta, deberá registrarse utilizando su RUT para verificar que se encuentre en el sistema. Si el RUT no está registrado en el SISTEMA, no podrá acceder a las funciones de la aplicación. Una vez validado, el usuario podrá completar sus datos personales para habilitar todas las herramientas.
Dentro de la aplicación, el usuario encontrará dos botones principales: "Agendar Hora" y "Mis Horas".

Agendar Hora: Al pulsar este botón, el usuario deberá elegir la especialidad requerida (Controles Médicos o Psicología). Tras seleccionar la especialidad, aparecerá un listado de las horas disponibles con sus respectivos profesionales en un rango de 1 a 2 días. En este paso, el usuario podrá elegir el tiempo de antelación para su recordatorio. Una vez confirmada la cita, se enviará un correo electrónico con los detalles del agendamiento: fecha, hora, profesional y box correspondiente. Por defecto, el sistema enviará una notificación al correo avisando una hora antes de la cita médica.

Mis Horas: Aquí el usuario podrá visualizar sus citas vigentes y el historial de agendamientos pasados. Al seleccionar una hora específica, se mostrarán los detalles y se permitirá modificar el tiempo del recordatorio según su preferencia.

Además, la aplicación cuenta con una sección de Perfil de Usuario, donde se podrán editar datos como el correo electrónico y la contraseña. Cabe destacar que el usuario no podrá modificar su nombre ni su apellido, ya que estos datos vienen predefinidos por el sistema.
